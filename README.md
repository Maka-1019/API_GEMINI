# API_GEMINI

Instrucciones para ejercutar el proyecto en otro equipo:
1. Crea el entorno virtual en el bash: phyton -m venv env
2. Activa el entorno virtual (Windows): .\env\Scripts\activate
3. Instala las dependencias: pip install -r .\requirements.txt
4. Ejecuta el proyecto: python .\app_gemini.py
